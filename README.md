# TPE: Artificial Intelligence (Archive & Modernization)

This repository is a hybrid project. It preserves an academic work completed in 2014 (High School TPE) while serving as a foundation for a modern web development.

## 🚀 Project Status

Currently, the project is in the **Preservation** phase.
The original site, exported from *Google Sites Classic*, is kept in its original state for historical reasons and fidelity to the initial content.

### Repository Structure

* `/legacy`: Contains the full 2014 site archive (HTML, CSS, Assets).
* `index.html`: A temporary entry point at the root that automatically redirects to the archive.
* `~`: (Coming soon) The site reconstruction using a modern stack.

## 🏛️ The Archive (2014)

The original site covered the evolution, types, and impacts of Artificial Intelligence. While the code generated by the tools of that era is now obsolete (proprietary Google scripts, non-responsive design), the textual content and research remain the core of the project.

**To view the archive directly:** [2014 site archive](https://chunhuynh.github.io/tpe-ia/legacy/home.html)

## 🛠️ Modernization Roadmap

The long-term goal is to transform this archive into a modern engineering portfolio:
1. **Extraction:** Retrieve raw content (Markdown).
2. **Refining:** Develop a responsive and accessible interface.
3. **Deployment:** Replace the root redirection with the new site while maintaining access to the `/legacy` folder.
